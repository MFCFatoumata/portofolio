01
Infrastructure Active Directory

Déploiement Active Directory & Services Réseau

Conception et déploiement complet d'un environnement Windows Server avec gestion centralisée des utilisateurs, ressources et droits d'accès.

  • Installation Windows Server 2022, configuration IP fixe, promotion en contrôleur de domaine
  • Structure OU (RH, IT, Finance, Commercial) avec méthode AGDLP
  • GPO : verrouillage session, blocage panneau de config, fond d'écran entreprise
  • Serveur de fichiers avec dossiers partagés cloisonnés par groupe
  • Intégration poste client Windows 10 au domaine, tests de permissions validés
  • Audit et journalisation activés (Event Viewer — Account Logon & Management)
Résultat : Accès cloisonnés validés — RH ne peut pas accéder aux dossiers IT et inversement. GPO appliquées. Audit actif sur le contrôleur de domaine.

Cloud & Identités Hybrides

Synchronisation AD ↔ Microsoft Entra ID (Azure)

Mise en place d'une infrastructure hybride complète — unification des identités on-premise et cloud avec authentification sécurisée.

  • Ajout du suffixe UPN public dans l'AD local et vérification DNS (enregistrement TXT)
  • Création d'utilisateurs en masse via fichier CSV (Bulk Create Azure)
  • Installation et configuration d'Azure AD Connect sur le contrôleur de domaine
  • Mise en place MFA (Microsoft Authenticator), SSO et Conditional Access
  • Synchronisation réussie : 21 utilisateurs locaux visibles dans Entra ID
Résultat : 21 utilisateurs synchronisés avec succès. Les comptes créés localement apparaissent automatiquement dans Azure avec le bon suffixe de domaine.
