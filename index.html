01
Infrastructure Hybride

Active Directory &
Microsoft Entra ID Sync

Conception et déploiement d'une infrastructure hybride complète : installation de Windows Server 2022, promotion en contrôleur de domaine, création d'une structure organisationnelle (OU, groupes AGDLP), synchronisation avec Microsoft Entra ID via Azure AD Connect, mise en place du MFA, SSO et Conditional Access. Résultat : Infrastructure 100% fonctionnelle permettant une authentification sécurisée unifiée on-premise et cloud. Les utilisateurs locaux apparaissent automatiquement dans Azure après synchronisation. Accès hybride opérationnel en environnement simulé entreprise.

02
Gestion des Identités & Sécurité

Infrastructure AD avec
Relation d'Approbation Forest

Mise en place d'une infrastructure AD complète : contrôleur de domaine DC1, serveur de fichiers SRV1, poste client PC1. Création d'une structure AGDLP, gestion des permissions par dossier partagé, déploiement de GPO (verrouillage session, blocage panneau de configuration, fond d'écran entreprise), puis établissement d'une relation d'approbation bidirectionnelle entre deux forêts AD distinctes.

Résultat : Cloisonnement des accès vérifié — les utilisateurs RH ne peuvent pas accéder aux dossiers IT et vice-versa. La relation d'approbation permet aux utilisateurs cross-domaine d'accéder aux ressources partagées. Audit et journalisation activés via GPO.
Windows Server 2022 AGDLP GPO Serveur de fichiers Approbation forêt Audit / Event Viewer PowerShell

04
Cloud & Identités Hybrides

Synchronisation AD
↔ Azure (Bulk & Suffixes)

Configuration avancée de la synchronisation hybride : ajout du suffixe UPN public dans AD local, vérification DNS via enregistrement TXT sur DNSExit, création d'utilisateurs en bloc sur Azure via fichier CSV, création d'un compte administrateur général Azure, connexion et configuration complète d'Azure AD Connect avec authentification MFA Authenticator.

Résultat : 21 utilisateurs synchronisés avec succès entre l'AD local et Microsoft Entra ID. Les comptes créés en local apparaissent automatiquement dans Azure avec le bon suffixe de domaine. Infrastructure hybride opérationnelle bout-en-bout.
